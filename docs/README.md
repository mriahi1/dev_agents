## Philosophy

**Cursor is the brain, toolkit provides the hands.**

This toolkit gives Cursor the ability to:
- Manage Linear tickets
- Create GitHub branches and PRs
- Execute DevOps operations

While Cursor provides:
- Code understanding
- Problem solving
- Implementation

And humans provide:
- Business context
- Quality assurance
- Final decisions
